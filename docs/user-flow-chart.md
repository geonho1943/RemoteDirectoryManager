## 문서 목적

- 원격파일메니저 서비스에서 사용자가 어떤 화면과 기능을 거쳐 이동하는지 한눈에 확인하기 위한 문서이다.
- 로그인, 탐색, 파일 작업, 태그 작업, 예외 흐름을 시각적으로 보여준다.
- 다이어그램은 Mermaid 지원 Markdown 뷰어 기준으로 작성되었다.

## 1. 전체 서비스 진입 플로우

```mermaid
flowchart TD
    A[사용자 서비스 접속] --> B{세션에 API Key 존재?}
    B -- 아니오 --> C[로그인 화면 진입]
    B -- 예 --> H[탐색 화면 진입 시도]

    C --> D[API Key 입력]
    D --> E[Health 확인]
    E --> F[루트 경로 목록 조회 시도]
    F --> G{인증 성공?}

    G -- 아니오 --> C1[오류 메시지 표시]
    C1 --> C
    G -- 예 --> H

    H --> I[세션 검증]
    I --> J{세션 유효?}
    J -- 아니오 --> C2[세션 정리 후 로그인 화면 이동]
    J -- 예 --> K[태그 목록 조회]
    K --> L[현재 경로 목록 조회]
    L --> M[탐색 화면 표시]
```

## 2. 탐색 화면 메인 플로우

```mermaid
flowchart TD
    A[탐색 화면 표시] --> B{사용자 선택}

    B --> C[경로 이동]
    B --> D[항목 선택]
    B --> E[새 폴더 생성]
    B --> F[파일 업로드]
    B --> G[숨김 파일 표시 전환]
    B --> H[새로고침]
    B --> I[연결 해제]

    C --> C1[브레드크럼 / 뒤로가기 / 상위폴더 / 더블클릭]
    C1 --> C2[대상 경로 목록 재조회]
    C2 --> A

    D --> D1[상세 정보 조회]
    D1 --> D2{선택 항목 유형}
    D2 -- 디렉터리 --> C
    D2 -- 파일 --> J[파일 작업 선택 가능]

    E --> E1[폴더명 입력]
    E1 --> E2[디렉터리 생성 API 호출]
    E2 --> E3[현재 경로 목록 재조회]
    E3 --> A

    F --> F1[파일 선택 또는 드래그앤드롭]
    F1 --> F2[충돌 정책 선택]
    F2 --> F3[업로드 API 호출]
    F3 --> F4[현재 경로 목록 재조회]
    F4 --> A

    G --> G1[includeHidden 값 변경]
    G1 --> G2[현재 경로 목록 재조회]
    G2 --> A

    H --> H1[현재 경로 목록 재조회]
    H1 --> A

    I --> I1[세션 API Key 제거]
    I1 --> I2[로그인 화면 이동]
```

## 3. 파일 선택 후 상세 작업 플로우

```mermaid
flowchart TD
    A[파일 선택] --> B[상세 정보 조회]
    B --> C[기본 미리보기 시도]
    C --> D{사용자 작업 선택}

    D --> E[Preview]
    D --> F[Stream]
    D --> G[Download]
    D --> H[Delete]
    D --> I[Tag 관리]

    E --> E1[파일 다운로드 endpoint 기반 미리보기]
    E1 --> B

    F --> F1[스트림 endpoint 기반 미리보기]
    F1 --> B

    G --> G1[다운로드 API 호출]
    G1 --> B

    H --> H1[삭제 확인]
    H1 --> H2[삭제 API 호출]
    H2 --> H3[선택 상태 초기화]
    H3 --> H4[현재 경로 목록 재조회]
    H4 --> K[목록 화면 복귀]

    I --> I1{태그 작업 유형}
    I1 --> I2[기존 태그 선택 추가]
    I1 --> I3[새 태그 입력 추가]
    I1 --> I4[연결된 태그 제거]

    I2 --> I5[태그 부여 API 호출]
    I3 --> I5
    I4 --> I6[태그 제거 API 호출]
    I5 --> I7[태그 목록 갱신]
    I6 --> I7
    I7 --> B
```

## 4. 업로드 충돌 처리 플로우

```mermaid
flowchart TD
    A[업로드 요청] --> B{같은 이름 파일 존재?}

    B -- 아니오 --> C[임시 파일 저장]
    B -- 예 --> D{충돌 정책}

    D -- FAIL --> E[업로드 실패 반환]
    D -- OVERWRITE --> F[기존 파일 백업 staging]
    D -- AUTO_RENAME --> G[새 이름 생성]

    G --> C
    F --> C
    C --> H[실제 저장 위치로 이동]
    H --> I[파일 메타데이터 동기화]
    I --> J{DB 반영 성공?}
    J -- 예 --> K[커밋 후 staging 정리]
    J -- 아니오 --> L[롤백 및 원상복구]
```

## 5. 예외 및 인증 실패 흐름

```mermaid
flowchart TD
    A[보호된 API 호출] --> B{API Key 유효?}
    B -- 아니오 --> C[401 Unauthorized]
    C --> D[클라이언트에서 세션 만료 처리]
    D --> E[로그인 화면 이동]

    B -- 예 --> F{요청 처리 성공?}
    F -- 예 --> G[정상 응답]
    F -- 아니오 --> H[오류 JSON 응답]
    H --> I[토스트 또는 오류 메시지 표시]
    I --> J{복구 가능한가?}
    J -- 예 --> K[현재 화면 유지 후 재시도]
    J -- 아니오 --> E
```

## 6. 화면별 선택 가능 기능 요약

- 로그인 화면
  - API Key 입력
  - 연결 시도

- 탐색 화면 공통
  - 브레드크럼 이동
  - 뒤로가기
  - 상위 폴더 이동
  - 새로고침
  - 숨김 파일 표시 전환
  - 테마 전환
  - 연결 해제

- 디렉터리 선택 시
  - 열기
  - 더블클릭 이동
  - 하위 항목 탐색
  - 현재 경로 기준 업로드
  - 현재 경로 기준 새 폴더 생성

- 파일 선택 시
  - 상세 정보 보기
  - Preview
  - Stream
  - Download
  - Delete
  - 태그 추가
  - 태그 제거
